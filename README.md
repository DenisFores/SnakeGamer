#  Snake 3D (Unity)

##  Описание проекта

Данный проект представляет собой 3D-прототип игры **Snake**, разработанный в Unity.
Игрок управляет змеёй, которая автоматически движется вперёд, может поворачивать, собирать еду и увеличивать длину хвоста.

В проекте реализованы:

* 3D-модель головы змеи (создана в Blender)
* Игровая механика движения
* Система хвоста
* Взаимодействие с едой
* Камера слежения
* Пользовательский интерфейс (Game Over, Restart, Exit)

---

##  Управление

| Действие       | Клавиша |
| -------------- | ------- |
| Поворот влево  | A / ←   |
| Поворот вправо | D / →   |

Змея движется вперёд автоматически.

---

##  Реализованные функции

###  Основная механика

* Постоянное движение вперёд
* Управление поворотом
* Камера, следующая за игроком

### Хвост

* Добавление сегментов при поедании еды
* Плавное следование сегментов
* Уменьшение размера каждого сегмента

###  Еда

* Объекты с триггером
* Взаимодействие со змеёй
* Увеличение хвоста

###  Система завершения игры

* Game Over при ошибке
* Замедление времени
* UI с кнопками Restart и Exit
